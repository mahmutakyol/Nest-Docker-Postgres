# You Need 
- NPM
- Node.js
- NestJS
- Vue.js
- Docker
- Postgres
- Socket.io

## For Back End
```bash
$ cd api  
$ docker-compose build  
$ docker-compose up
````

## For Front End
```bash
$ cd app
$ docker-compose build
$ docker-compose up
```

> after the running you can go `localhost:8081` on your browser
